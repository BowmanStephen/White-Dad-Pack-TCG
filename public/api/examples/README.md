# DadDeck API - Code Examples

This directory contains example code for integrating with the DadDeck API.

## JavaScript Examples

All JavaScript examples use modern async/await syntax and work in both Node.js and browser environments.

### Installation

```bash
npm install node-fetch
```

## Python Examples

All Python examples require Python 3.7+ and the `requests` library.

### Installation

```bash
pip install requests
```
