## Free Sound Sources

- [soundsnap.com](https://soundsnap.com) - High quality Asian instruments

## Audio Guidelines

- Keep volumes normalized (peak at -3dB)
- Fade in/out to avoid clicks
- Target 44.1kHz, 128-192kbps MP3
- Keep file sizes under 100KB each for fast loading
